# Password Generator using Python

- Description
This is a beginner-friendly Python project that generates strong and random passwords based on user-defined length.  
It uses Python's built-in libraries to include letters, numbers, and special characters.

## Technologies Used
- Python
- random module
- string module

## Features
- Generates strong and secure passwords
- User can choose password length
- Includes uppercase, lowercase, numbers, and symbols
- Simple console-based interface

## Learning Outcome
- Learned how to use Python modules
- Improved logic building
- Understood basics of password security
- Practiced exception handling
